SEBS — Satışa hazırlık checklist (Notion içe aktarma)
====================================================

Dosya: satis-hazirlik-checklist-notion.csv

Adımlar:
1. Notion'da yeni bir sayfa açın → … menü → Import
2. "CSV" seçin → satis-hazirlik-checklist-notion.csv dosyasını yükleyin
3. "Import as" → Table / Database (veritabanı) seçin
4. İlk sütun "Görev" → Title (başlık) olarak ayarlayın
5. Önerilen özellik tipleri:
   - Öncelik → Select (P0, P1, P2)
   - Durum → Select (Tamamlandı, Kısmen, Bekliyor)
   - Sorumlu → Select veya Text
   - Tahmini gün → Text
   - Kategori → Select
   - ID → Text
6. Görünüm ekleyin: "P0 Bekliyor" filtresi (Öncelik = P0, Durum = Bekliyor)

Excel için: satis-hazirlik-checklist.xlsx dosyasını doğrudan Excel veya Google Sheets ile açın.

Güncelleme: python3 scripts/export_satis_hazirlik_checklist.py
(önce: .venv-checklist/bin/pip install openpyxl veya proje venv)
