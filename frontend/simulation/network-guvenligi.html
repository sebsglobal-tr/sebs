Temel Network Simülasyonu

SEBS Network Lab 1: Küçük Ofis Ağı Kurulumu ve WAN Bağlantısı

Bu senaryoda önce ofis içi LAN'ı kurar, sonra aynı ağı WAN'a bağlarsınız. Hedef; cihaz konumlandırma, IP adresleme, gateway mantığı ve uzak ağ erişimini uygulamalı öğrenmektir.

Simülasyonlara Dön

Aşama 1 — LAN Kurulumu

1) 1 adet switch ekleyin
2) 3 adet PC ekleyin
3) 1 adet yazıcı ekleyin
4) 1 adet yerel server ekleyin
5) Tüm LAN cihazlarını switch'e bağlayın

Aşama 2 — WAN Bağlantısı

6) Şirket router'ı ekleyin
7) ISP / WAN cloud ekleyin
8) Uzak sunucu ekleyin
9) LAN → Router → WAN → Uzak Sunucu zincirini kurun
10) Gateway ve erişim testini tamamlayın
